## DOWAZO

### Subject : Safety 방범 로봇

### 시나리오
1. 터틀봇이 주행을 함

2. 주행을 하다가 주변에 사람을 발견함    // 사람 디텍팅

3. 사람의 얼굴, 상의, 하의 분리        // 얼굴 인식, 상의 인식, 하의 인식

4. 입력된 정보와 비교

   4-1. 하나라도 틀린 경우 다시 사람들을 확인하러 다님

   4-2. 다 맞은 경우 사람을 찾는 것을 멈추고, 위치 정보를 서버에 보냄.
